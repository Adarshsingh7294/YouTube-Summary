# YouTube Summary Generator

> Paste a YouTube link. Get a short summary, a detailed breakdown, key takeaways, and timestamped highlights ‚Äî generated by AI and exportable to PDF.

Built with **Next.js 15 (App Router)**, **TypeScript**, **Tailwind CSS**, and the **OpenAI SDK**. Works against any OpenAI-compatible endpoint (OpenAI, Groq, OpenRouter, Ollama, LM Studio, etc.).

---

## Features

- Paste a YouTube URL (`youtube.com/watch`, `youtu.be`, `/shorts`, `/embed`)
- Automatic transcript fetching
- Three summary views in one pass:
  - **Short** ‚Äî 2-3 sentence TL;DR
  - **Detailed** ‚Äî 300-500 word breakdown
  - **Key Takeaways** ‚Äî bulleted actionable insights
- Auto-generated **timestamps** with click-to-open on YouTube
- **Copy** the current view to your clipboard
- **Download as PDF** ‚Äî beautifully typeset, multi-page
- **Dark mode** (system-aware, persisted)
- **Responsive** layout (mobile to desktop)
- **Loading states** with stage progress
- **Error handling** with clear messages

## Limitations

- Relies on YouTube transcripts being publicly available. Private,
  members-only, or non-captioned videos will return a 404.
- Long videos may be truncated to fit the model context window
  (the head and tail of the transcript are kept, the middle dropped).
- Generation time scales with transcript length; expect 15-60 seconds for
  average videos.
