# YouTube Summary Generator

> Paste a YouTube link. Get a short summary, a detailed breakdown, key takeaways, and timestamped highlights — generated by AI and exportable to PDF.

Built with **Next.js 15 (App Router)**, **TypeScript**, **Tailwind CSS**, and the **OpenAI SDK**. Works against any OpenAI-compatible endpoint (OpenAI, Groq, OpenRouter, Ollama, LM Studio, etc.).

---

## Features

- Paste a YouTube URL (`youtube.com/watch`, `youtu.be`, `/shorts`, `/embed`)
- Automatic transcript fetching
- Three summary views in one pass:
  - **Short** — 2-3 sentence TL;DR
  - **Detailed** — 300-500 word breakdown
  - **Key Takeaways** — bulleted actionable insights
- Auto-generated **timestamps** with click-to-open on YouTube
- **Copy** the current view to your clipboard
- **Download as PDF** — beautifully typeset, multi-page
- **Dark mode** (system-aware, persisted)
- **Responsive** layout (mobile to desktop)
- **Loading states** with stage progress
- **Error handling** with clear messages

## Quick Start

```bash
# 1. Install
npm install

# 2. Configure
cp .env.example .env.local
# then edit .env.local with your API key

# 3. Run
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

See **[SETUP.md](./SETUP.md)** for the full step-by-step guide (including
alternatives like Groq, OpenRouter, and local Ollama).

## Environment Variables

| Variable | Required | Default | Description |
| --- | --- | --- | --- |
| `OPENAI_API_KEY` | Yes | — | API key for your provider |
| `OPENAI_BASE_URL` | No | `https://api.openai.com/v1` | Any OpenAI-compatible endpoint |
| `OPENAI_MODEL` | No | `gpt-4o-mini` | Model name to use |

### Provider examples

```bash
# OpenAI
OPENAI_API_KEY=sk-...
OPENAI_BASE_URL=https://api.openai.com/v1
OPENAI_MODEL=gpt-4o-mini

# Groq
OPENAI_API_KEY=gsk_...
OPENAI_BASE_URL=https://api.groq.com/openai/v1
OPENAI_MODEL=llama-3.1-70b-versatile

# OpenRouter
OPENAI_API_KEY=sk-or-...
OPENAI_BASE_URL=https://openrouter.ai/api/v1
OPENAI_MODEL=meta-llama/llama-3.1-8b-instruct:free

# Ollama (local)
OPENAI_API_KEY=ollama
OPENAI_BASE_URL=http://localhost:11434/v1
OPENAI_MODEL=llama3.1
```

## Deployment (Vercel)

1. Push to GitHub.
2. Import the repo in [Vercel](https://vercel.com/new).
3. Add the three env vars from above in **Project Settings → Environment Variables**.
4. Deploy. The `/api/summarize` route runs on the Node.js runtime with a 5-minute max duration.

For other providers, you may need to allow outbound requests to that host
(and for self-hosted models, expose the model to Vercel's network).

## Limitations

- Relies on YouTube transcripts being publicly available. Private,
  members-only, or non-captioned videos will return a 404.
- Long videos may be truncated to fit the model context window
  (the head and tail of the transcript are kept, the middle dropped).
- Generation time scales with transcript length; expect 15-60 seconds for
  average videos.
